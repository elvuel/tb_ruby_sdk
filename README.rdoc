== 淘宝开放平台 ruby sdk 2.0-pre
	* 淘宝开放平台ruby版sdk。

== 如何使用?
	* 请自行安装好ruby环境
	* 安装Crack, 位于http://github.com/jnunemaker/crack
	* 在设置 base_setting.rb 中，设定你所申请的APP_KEY及APP_SECRET
	* 添加所需要使用的开放平台api到api.rb文件中（请参照已有示例，另外 params中 *字符为必需项 !字符为上传）
